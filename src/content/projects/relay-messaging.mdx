## Context

Relay set out to build a messaging app for teams that doesn't create anxiety. The core insight: most messages don't need immediate responses, but most apps make them feel urgent.

<PlaceholderImage aspect="16/9" caption="Async-first messaging — no read receipts, no typing indicators" />

## Design Decisions

Messages default to async — no read receipts, no typing indicators. Users opt into urgency, not out of it. A "focus mode" batches notifications into digests at intervals the user chooses.

<PlaceholderImage aspect="3/2" caption="Focus mode — notification digest settings" />

## Results

- 82% of users reported lower communication anxiety
- Team response quality improved (measured via manager surveys)
- 4.6 star average rating across platforms
